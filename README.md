UIBuilder-PHP
========================

UIBuilderPHP es una librería PHP para soporte de clases dinamicas que pueden interactuar con librerias o paquetes de frontend como objetos json.

## Instalación

``` bash
$ composer require maximosojo/uibuilder-php
```
